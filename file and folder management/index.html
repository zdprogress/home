Creating a folder

If you use a computer weather at work or home, you must have the knowledge of working with folders because anything stored on a computer is located in in a certain folder. Therefor it will require you to know how

  • To create folders for storing differnt files and subfolders, a subfolder is any folder within onather folder for example if i create a folder called School and inside it i create another folder called Senior four work then Senior four work becomes a subfolder of School
  • To navigate through folders, for example knowing how to open a folder, how to go back one folder and how to close a folder
  • To delete a folder. Sometimes you may have some unnecessary folders that you may want to delete, these might be empty folders which bring confusion yet they are useless

Follow the these steps to create a folder

  1. Right click where you want to create a folder for example on the desktop or inside another folder, make sure you right click in an empty space
  2. On the dialog box that appears point or click on New then Folder
  3. A new folder will apper, but before doing anything else first type the name of the folder for this case we have named it Derrick and hit the enter key on your keyboard
  4. Your new folder will look like the one bellow

Renaming a folder

  1. Right click the folder you want to rename
  2. On the dialogue box that appears click rename
  3. Type a new name for the folder and hit enter on the keyboard for this case we provided Zziwaas the folder's new name

Deleting a folder

Follow this steps to delete a folder

  1. Right click on the folder you want to delete, on the dialogue box that appears click delete This can also work when deleting a file just right click on the file you want to delete and then click delete on the dialogue box that appears
