Basic HTML Structure

Every HTML document follows a fundamental structure. Understanding this structure is essential for creating well-formed web pages.

The Essential Elements:

  1. <!DOCTYPE html>:

    This declaration is the very first thing in your HTML document. It tells the browser which version of HTML the page is using. For HTML5, it's simply <!DOCTYPE html>.

  2. <html>:

    This is the root element of an HTML page. It wraps all other elements in the document. The lang attribute is used within the opening html tag, like this: <html lang="en"> This specifies the language of the document.

  3. <head>:

    This section contains meta-information about the HTML document. It's not displayed on the page itself. It includes things like the page title, character set, and links to external files (CSS, JavaScript).

  4. <title>:

    Defines the title of the document, which appears in the browser's title bar or tab. It's also used by search engines.

  5. <meta>:

    Provides metadata about the HTML document. Examples: charset="UTF-8": Specifies the character encoding. name="viewport": Configures the viewport for responsive design.

  6. <body>:

    This section contains the visible content of the HTML page. Everything you want to display on the page goes inside the <body>.

Example:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>My First HTML Page</title>
        </head>
        <body>
            <h1>Hello, World!</h1>
            <p>This is my first HTML page.</p>
        </body>
        </html>

Explanation of the example:

  • The <!DOCTYPE html> declaration tells the browser it's an HTML5 document.
  • The <html> tag specifies that it is a html document, and the lang="en" attribute specifies the language is English.
  • The <head> section includes the character set, viewport settings, and the page title.
  • The <body> section contains a heading (<h1>) and a paragraph (<p>), which will be displayed in the browser.

Key Takeaways:

  • Every HTML document should have this basic structure.
  • The <head> contains meta-information, and the <body> contains the visible content.
  • The lang attribute inside the html tag, is a good practice to include.
